Ember's Unofficial F-Droid Repo

This repository allows F-Droid users to install uGet and Signal on their device without going through Google Play. These apks are sourced pre-compiled from the respective app developers and are not compiled by the repo maintainer.

To begin, scan the QR code above, tap the following link, or enter the following URL into F-Droid:

https://fdroid.heartshine.xyz/repo

Software offered:

  • uGet: a powerful multi-threaded download manager. (Homepage, Sourceforge)
  • Signal: encrypted chat, video and audio calling, group messages and more for the privacy-conscious. This repo distributes Twin Helix's de-Google'd Signal fork. (Signal homepage, Twin Helix homepage, GitHub)

**I have noticed increased traffic coming from Tor endpoints lately, and have adjusted my Cloudflare firewall to accommodate. Tor users should no longer be blocked from this repo.
